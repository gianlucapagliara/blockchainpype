# BlockchainPype

A Python library for interacting with multiple blockchain networks with a unified interface.

## Features

- Multi-chain support (EVM-compatible chains and Solana)
- Wallet management with secure key handling
- Asset operations and DApp integrations
- Transaction handling with gas estimation
- Block explorer integration

## Installation

```bash
pip install blockchainpype
```
